# Glossary

## Terms

### Account
A data structure representing a user or contract on the network.

### Block
A collection of transactions with associated metadata.

### Consensus
The process of achieving agreement on block ordering.

### Finality
The cryptographic guarantee that a block cannot be reverted.

### Gas
A unit measuring computational cost.

### Validator
A network participant that secures the network through consensus.
